# HongCafe Agent Office MVP

홍카페용 AI 에이전트 가상 사무실의 첫 번째 MVP입니다.

## 포함 기능

- 서버 연결 준비 UI
- PM / 개발 / QA 에이전트 시각화
- 픽셀풍 가상 사무실 레이아웃
- 업무 지시 입력과 라이브 로그
- 에이전트 클릭 상세 보기
- 서브 에이전트 추가
- 리뷰 라운드 시뮬레이션

## 현재 상태

이 버전은 프론트엔드 시뮬레이션입니다. 실제 가상서버와 붙일 때는 아래 구성이 필요합니다.

- 백엔드 오케스트레이터 API
- WebSocket 또는 SSE 이벤트 스트림
- 작업 큐 저장소
- 모델 호출 레이어
- 인증/권한 처리

## 다음 권장 단계

1. `server/` 폴더를 만들어 연결 API를 추가합니다.
2. `connectServer()`를 실제 헬스체크 요청으로 교체합니다.
3. `dispatchMission()`를 서버의 작업 생성 API 호출로 교체합니다.
4. 라이브 로그를 실시간 이벤트 구독으로 전환합니다.
5. Git 작업/배포 로그를 에이전트 카드와 연결합니다.
